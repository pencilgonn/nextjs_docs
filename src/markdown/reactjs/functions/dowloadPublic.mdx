# dowloadPublic()

The downloadPublic function you provided is designed to download a file from a public URL. It fetches the file using the fetch API, converts the response into a blob, creates a URL for the blob, and triggers a download by creating a temporary link element with the appropriate attributes.

## Code

```downloadPublic.ts

const onDowloadPublic = async (url: string) => {
    try {
        const response = await fetch(url);
        const blob = await response.blob();
        const blobUrl = URL.createObjectURL(blob);

        const link = document.createElement("a");
        link.href = blobUrl;
        link.download = "name_file.ext"
        document.body.appendChild(link);
        link.click();
        document.body.removeChild(link);
        URL.revokeObjectURL(blobUrl);
    } catch (error) {
        console.error("Error fetching or parsing file:", error);
    }
};
```
